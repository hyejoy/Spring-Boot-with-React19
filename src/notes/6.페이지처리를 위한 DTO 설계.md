# Spring Boot: PageResponseDTO & 목록 처리

## 목록 처리에 필요한 데이터

화면에서 페이징 목록을 보여주려면 단순히 데이터 목록만으로는 부족함.

| 데이터 | 설명 |
|--------|------|
| `dtoList` | 현재 페이지의 목록 데이터 |
| `current` | 현재 페이지 번호 |
| `pageNumList` | 화면에 표시할 페이지 번호 목록 (예: [11, 12, 13 ... 20]) |
| `pageRequestDTO` | 요청 정보 (page, size, 검색 조건 등) |
| `prev` / `next` | 이전/다음 페이지 존재 여부 (boolean) |
| `totalCount` | 전체 데이터 개수 |
| `prevPage` / `nextPage` | 이전/다음 그룹의 첫 페이지 번호 |
| `totalPage` | 전체 페이지 수 |

## PageResponseDTO

```java
@Data
public class PageResponseDTO<E> {  // 제네릭 — 나중에 다른 DTO 타입도 사용 가능

    private List<E> dtoList;
    private List<Integer> pageNumList;
    private PageRequestDTO pageRequestDTO;
    private boolean prev, next;
    private int totalPage, current, prevPage, nextPage, start, end;
    private long totalCount;

    @Builder(builderMethodName = "withAll")
    public PageResponseDTO(List<E> dtoList, PageRequestDTO pageRequestDTO, long totalCount) {
        this.dtoList = dtoList;
        this.pageRequestDTO = pageRequestDTO;
        this.totalCount = totalCount;

        // 전체 페이지 수
        this.totalPage = (int) Math.ceil(totalCount / (double) pageRequestDTO.getSize());

        // 현재 페이지
        this.current = pageRequestDTO.getPage();

        // 끝 페이지 번호 계산
        // 예: 7페이지 → ceil(7/10.0) * 10 = 10
        //     11페이지 → ceil(11/10.0) * 10 = 20
        int end = (int) (Math.ceil(current / 10.0)) * 10;

        // 시작 페이지 번호 = 끝 페이지 - 9
        int start = end - 9;

        // 실제 마지막 페이지가 end보다 작으면 조정
        int last = (int) Math.ceil(totalCount / (double) pageRequestDTO.getSize());
        end = Math.min(end, last);

        this.start = start;
        this.end = end;

        // 이전/다음 여부
        this.prev = start > 1;
        this.next = totalCount > (long) end * pageRequestDTO.getSize();

        // 이전/다음 그룹 첫 페이지
        this.prevPage = start - 1;
        this.nextPage = end + 1;

        // 화면에 표시할 페이지 번호 리스트 [start ... end]
        this.pageNumList = IntStream.rangeClosed(start, end)
            .boxed()
            .collect(Collectors.toList());
    }
}
```

### 핵심 페이지 번호 계산 공식

```
현재 7페이지  → ceil(7/10.0) = 1 → 1 * 10 = 10 (끝) → 10 - 9 = 1 (시작)
현재 10페이지 → ceil(10/10.0) = 1 → 1 * 10 = 10 (끝) → 10 - 9 = 1 (시작)
현재 11페이지 → ceil(11/10.0) = 2 → 2 * 10 = 20 (끝) → 20 - 9 = 11 (시작)
```

## Service 구현 — list()

```java
@Override
public PageResponseDTO<TodoDTO> list(PageRequestDTO pageRequestDTO) {

    // 1. Pageable 생성 (페이지 번호 0부터 시작 주의!)
    Pageable pageable = PageRequest.of(
        pageRequestDTO.getPage() - 1,  // 1페이지 → 0
        pageRequestDTO.getSize(),
        Sort.by("tno").descending()
    );

    // 2. DB 조회
    Page<Todo> result = todoRepository.findAll(pageable);

    // 3. Entity → DTO 변환
    List<TodoDTO> dtoList = result.getContent().stream()
        .map(todo -> modelMapper.map(todo, TodoDTO.class))
        .collect(Collectors.toList());

    // 4. PageResponseDTO 생성
    return PageResponseDTO.<TodoDTO>withAll()
        .dtoList(dtoList)
        .pageRequestDTO(pageRequestDTO)
        .totalCount(result.getTotalElements())
        .build();
}
```

## 흐름 정리

```
PageRequestDTO (요청)         PageResponseDTO (응답)
  - page: 11                    - dtoList: [Todo11 ... Todo20]
  - size: 10         →          - pageNumList: [11, 12 ... 20]
                                - prev: true (prevPage: 10)
                                - next: true (nextPage: 21)
                                - totalCount: 51712
                                - totalPage: 5172
```

    log.info("전체 데이터: " + response.getTotalCount());
